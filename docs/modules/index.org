*** [[file:reel/index.org][Reel Module]] - Programmatic Video Creation
:PROPERTIES:
:CUSTOM_ID: reel-module---programmatic-video-creation
:END:
Declarative, code-driven video: define a composition of timed, animated layers,
then render it frame-by-frame to GIF, an image sequence, or MP4/WebM.  Fully
headless except for the optional live preview player.

- *[[file:reel/composition.org][LrgReel]]* - the composition (id, resolution, fps, duration, clip stack, props)
- *[[file:reel/timing.org][LrgReelContext / LrgReelSequence]]* - the relative-frame model: shift, series, loop, freeze
- *[[file:reel/animation.org][interpolate / spring]]* - range interpolation and spring physics
- *[[file:reel/clips.org][LrgReelClip]]* - timed, opacity'd renderable layers (vfunc or callback)
- *[[file:reel/rendering.org][LrgReelRenderer]]* - the offline frame-by-frame engine
- *[[file:reel/exporters.org][LrgReelExporter]]* - GIF, image-sequence, and ffmpeg MP4/WebM output
- *[[file:reel/transitions.org][LrgReelTransition]]* - CPU fade / wipe / dissolve / slide
- *[[file:reel/audio.org][LrgReelAudioTrack]]* - offline audio mixing and muxing
- *[[file:reel/player.org][LrgReelPlayer]]* - interactive play/scrub preview window

prefabs. See [[file:../editor/overview.org][Editor Overview]] for the full guide.

- *[[file:../editor/level-model.org][LrgLevel / LrgNode / LrgNodeVisual]]* - authoring document and node tree
- *[[file:../editor/yaml-format.org][.rlevel Format]]* - YAML serialization schema
- *[[file:../editor/editor-runtime.org][LrgEditor / Commands / Selection / Host]]* - editor runtime and embedding
- *[[file:../editor/prefabs.org][Prefabs]]* - reusable saved node subtrees
- *[[file:../editor/projects-and-assets.org][LrgProject / LrgAssetDatabase]]* - project manifest and asset scanning
- *[[file:../editor/tutorial.org][Tutorial]]* - build a level in C step by step

** Building with Multiple Modules
:PROPERTIES:
:CUSTOM_ID: building-with-multiple-modules
:END:
Once available, use modules in concert:

#+begin_src C
/* Core: Setup engine */
LrgEngine *engine = lrg_engine_get_default ();
lrg_engine_startup (engine, &error);

/* ECS: Create game world */
LrgWorld *world = lrg_world_new ();

/* Input: Handle player input */
LrgInputMap *input = lrg_engine_get_input_map (engine);

/* Tilemap: Load level */
LrgTilemap *map = lrg_data_loader_load_file (loader, "level1.yaml", &error);

/* Physics: Simulate */
LrgPhysicsWorld *physics = lrg_world_get_physics (world);

/* Audio: Play sounds */
LrgAudioManager *audio = lrg_engine_get_audio_manager (engine);

/* Save: Persist game state */
LrgSaveManager *save = lrg_engine_get_save_manager (engine);
#+end_src
