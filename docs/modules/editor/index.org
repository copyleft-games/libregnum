* Editor Module Reference
:PROPERTIES:
:CUSTOM_ID: editor-module-reference
:END:
API reference for the editor and level-authoring subsystem.
Requires =-DLRG_BUILD_EDITOR= (=BUILD_EDITOR ?= 1=).

** Core Types
:PROPERTIES:
:CUSTOM_ID: core-types
:END:
| Type | Kind | Description |
|------+------+-------------|
| =LrgLevel= | Final | Serializable authoring document (=.rlevel= file) |
| =LrgNode= | Derivable | Node in the editable scene tree; 2D/3D unified |
| =LrgNodeVisual= | Final | Tagged visual payload for a node |
| =LrgComponentDesc= | Final | Serializable component description (type name + props) |
| =LrgScriptBinding= | Final | Serializable script reference (language + path) |
| =LrgLevelSerializer= | Final | =.rlevel= / =.rprefab= YAML serializer |

** Bridge Functions
:PROPERTIES:
:CUSTOM_ID: bridge-functions
:END:
| Function | Signature | Description |
|----------+-----------+-------------|
| =lrg_level_to_scene= | =LrgLevel *→ LrgScene *= | Bake primitive-visual nodes to =LrgScene= |
| =lrg_level_instantiate= | =LrgLevel, LrgWorld, LrgEngine, GError **→ gboolean= | Populate =LrgWorld= from level |
| =lrg_level_from_scene= | =LrgScene *→ LrgLevel *= | Import Blender scene as editable level |

** Editor Runtime Types
:PROPERTIES:
:CUSTOM_ID: editor-runtime-types
:END:
| Type | Kind | Description |
|------+------+-------------|
| =LrgEditor= | Final | Host-agnostic editor runtime; owns level, selection, undo stack |
| =LrgEditorCommand= | Final | Discriminated undoable mutation |
| =LrgEditorSelection= | Final | Multi-select set with primary node |
| =LrgEditorHost= | Interface | Embedding contract (render target, camera, input) |
| =LrgEditorSoftwareHost= | Final | Concrete data-holding host implementation |

** Project and Asset Types
:PROPERTIES:
:CUSTOM_ID: project-and-asset-types
:END:
| Type | Kind | Description |
|------+------+-------------|
| =LrgProject= | Final | =project.ryaml= manifest; asset dirs; game output |
| =LrgAssetDatabase= | Final | Recursive asset scanner + classifier |
| =LrgAssetEntry= | Final | Single classified asset record (path, name, type, guid) |

** Prefab Functions
:PROPERTIES:
:CUSTOM_ID: prefab-functions
:END:
| Function | Description |
|----------+-------------|
| =lrg_prefab_clone= | Deep copy of a node subtree with fresh guids |
| =lrg_prefab_instantiate= | Produce a placeable copy of a prefab |
| =lrg_prefab_save= | Save a node subtree as a =.rprefab= file |
| =lrg_prefab_load= | Load a prefab from a =.rprefab= file |
